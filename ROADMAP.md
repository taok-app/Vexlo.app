# Vexlo AI Roadmap

Vexlo AI is an open-source AI Engineering Workspace. This roadmap outlines our planned features and improvements.

## Current Release: v1.0.0

Focus on core functionality and stability.

### Completed (v1.0.0)
- Multi-agent AI support
- User authentication and task management
- Vercel Sandbox integration
- Git automation
- Real-time task monitoring
- Comprehensive error handling

## Upcoming: v1.1 (Q3 2026)

### Features
- Enhanced compare mode for multi-agent execution
- Advanced git workflows (rebase, cherry-pick, merge)
- More AI model options (Claude 3.5 Sonnet, GPT-4, etc.)
- Improved deployment logging and feedback
- Custom agent templates
- Webhook support for automations

### Infrastructure
- Better deploy error messaging
- Improved performance metrics
- Enhanced security audit logging
- Optimized sandbox resource allocation

## Future: v2.0

### Vision
- Team collaboration and shared workspaces
- Advanced project templates
- Custom LLM integrations
- Agent orchestration
- Audit trails and compliance features
- Enterprise deployment options
